# simple-photo-manager

App to organize Your photo collection!

## Basic usage

``` bash
python3 simple-photo-manager.py --config <CONFIG_FILE>
```

## Example config file

``` json
{
    "file-scheme": "IMG_{}.JPG",
    "action": "copy",
    "from": "from",
    "to": "to",
    "ids": [
        1118,
        1129,
        1141,
        1170,
        1209,
        1274,
        1292,
        1306
    ]
}
```
